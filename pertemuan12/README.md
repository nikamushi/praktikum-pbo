# [INF2413] Proyek SOLID OOP Python

## Pertemuan 12: Dokumentasi dan Version Control

## Deskripsi Proyek

Proyek ini mendemonstrasikan implementasi prinsip SOLID (SRP, OCP, DIP) pada sistem checkout dengan menggunakan Abstraction (Interface) dan Depedency Injection.

### Struktur File

- `refactor_solid.py` : Kode inti yang sudah direfaktor dan ditambahkan logging.
- `README.md` : Dokumen ini.

### Cara Menjalankan

1. Pastikan Python 3.x terinstal.
2. Jalankan file utama dari terminal.
   ```bash
   python refactor_solid.py
   ```
